## Features

- **Food tracking** — Log meals manually or in natural language ("I had a chicken sandwich"). The AI detects calories, macros and meal type automatically.
- **Recipe generation** — Provide your ingredients and serving count, Mistral generates 3 detailed recipes with steps, cooking times and nutritional values.
- **Weekly routine** — 7-day meal plan generated by AI based on your calorie goals, difficulty preferences, budget and dietary restrictions.
- **Nutrition analysis** — Track macros (protein, carbs, fats, fiber) with personalized advice based on your goal (lose, maintain, gain).
- **Smart suggestions** — Recommendations based on your favorite recipes and remaining calorie budget.
- **Daily recipes** — Fresh recipes generated automatically every morning (5 AM cron job).
- **Cooking mode** — Step-by-step instructions with progress bar.
- **Notifications** — Customizable reminders for each meal.
- **Themes** — Light, dark nature, dark neutral.

## Getting Started

### Prerequisites

- Node.js (LTS)
- [pnpm](https://pnpm.io/)
- Docker & Docker Compose
- A [Mistral API key](https://console.mistral.ai/)

### Setup

```bash
# Clone the repo
git clone https://github.com/daniilsys/kerak.git
cd kerak

# Install dependencies
pnpm install

# Set up environment variables
cp apps/api/.env.example apps/api/.env
# Edit apps/api/.env with your Mistral key

# Start PostgreSQL
docker compose up -d

# Run migrations
cd apps/api
pnpm exec prisma migrate dev

# Import the CIQUAL ingredient database (optional)
pnpm import:ingredients
```

### Running

```bash
# Terminal 1 — API
pnpm dev:api

# Terminal 2 — Mobile app
pnpm dev:mobile
```

Expo starts on the default port. Scan the QR code with Expo Go or launch an emulator.

<details>
<summary><strong>Food Tracking</strong></summary>

| Method | Route                     | Description                 |
| ------ | ------------------------- | --------------------------- |
| POST   | `/logs`                   | Add a meal                  |

</details>
